## Install Jekyll on Windows Linux Subsystem (Ubuntu)

### Get Ubuntu up to date and Install Ruby

```bash
sudo apt-get update -y && sudo apt-get upgrade -y
sudo apt-get install -y build-essential ruby-full
```
